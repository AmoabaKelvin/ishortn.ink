We have completely redesigned the iShortn dashboard and landing page. Every page has been rebuilt with a refined design system focused on clarity, consistency, and ease of use.

## What's changed?

The entire interface has been overhauled with a minimal, modern design language. The goal was simple: make everything cleaner, more readable, and easier to navigate.

### Dashboard

- **Refined layout** — Tighter spacing, cleaner cards, and improved visual hierarchy across all pages
- **Better navigation** — Streamlined sidebar with a more intuitive workspace switcher
- **Improved analytics** — Cleaner charts, better data presentation, and more readable stats
- **Links management** — Redesigned link cards, edit drawer, and bulk action bar
- **Domains** — Cleaner domain cards with improved DNS configuration views
- **Folders** — Refreshed folder cards and settings modals
- **QR codes** — Updated creation flow and card designs
- **Settings** — Simplified settings pages with better form layouts
- **Teams** — Cleaner member management and team settings

### Landing page

- **New hero section** — Refined messaging with better visual flow
- **Updated features section** — Cleaner feature showcase
- **Pricing** — More readable pricing cards
- **Testimonials** — Refreshed testimonial layout
- **Changelog** — Improved changelog page and timeline design

### Brand identity

- **New wordmark font** — Updated the iShortn brand text to Eagle Lake for a more distinctive, memorable identity
- **Consistent typography** — Dedicated brand font utility applied across header, footer, and sidebar

## Design principles

This redesign follows a few core principles:

- **Less is more** — Removed visual noise and unnecessary decoration
- **Consistency** — Unified spacing, colors, and typography across every page
- **Readability** — Improved contrast, font sizes, and content hierarchy
- **Speed** — Fewer visual elements means faster rendering and less cognitive load

The result is an interface that feels lighter, faster, and more professional. We hope you enjoy using it as much as we enjoyed building it.
